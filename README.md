# 🍕 Pizza Sales Dashboard

## 📌 Project Overview
This Power BI dashboard analyzes pizza sales performance to identify revenue trends, peak order timings, top-selling pizzas, and customer preferences.

## 🎯 Objective
To help restaurant owners optimize:
- staffing during peak hours
- inventory planning
- menu strategy
- revenue growth

## 🛠 Tools Used
- Power BI
- Excel
- AI Tools
- Data Cleaning
- Data Visualization

## 📊 Key KPIs
- Total Revenue
- Total Orders
- Average Order Value
- Peak Order Time
- Best Selling Pizza
- Worst Selling Pizza

## 🔍 Insights
- Highest orders during evening hours
- Weekend sales outperform weekdays
- Certain pizza categories contribute maximum revenue
- Peak demand helps optimize delivery planning
